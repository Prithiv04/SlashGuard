# SlashGuard
PVMShield is a trustless validator slashing insurance protocol built on Polkadot's Paseo Asset Hub testnet. Stake DOT on validator outcomes using PVM precompiles for oracle-free resolution. Live contract: 0x736985ed65A72b1b44B572Ff75EB52Dd7d624eF9. Production-ready Web3 dApp with real TVL reads and storage proofs.
